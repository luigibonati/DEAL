# Examples

These minimal examples show the two common DEAL workflows. Both use incremental
selection with a target of 140 structures.

1. [N₂ dissociation with uncertainty preselection](1_n2_with_preselection/README.md)
2. [Formate dehydrogenation without preselection](2_formate_without_preselection/README.md)

Each example provides a minimal `input.yaml` and the equivalent command-line
invocation.
